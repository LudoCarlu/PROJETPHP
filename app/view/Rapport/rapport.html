Ce que l'on peut faire sur le site

  • Faire une demande d'inscription
  • Posséder son propre compte utilisateur si l'admin vous accepte
  • Consulter des artistes et/ou des albums
  • Ajouter à la base du site un artiste ou un album
  • Exprimer un commentaire sur un album
  • Noter un album (une seule note, modifiable)

Faire des recherches :

  • Sur un album par artiste, genre, date, note
  • Sur un utilisateur : Grace a son pseudo
    (s'il a mit un commentaire ou une note cela s'affichera dans sa description sinon ne s'affiche pas)

Pour les connexions (Seulement si l'utilisateur est connecté)

  • Voir l'actualité et un palmares sur la page index
  • Pouvoir voir les listes d'albums et acceder à leur description
  • Pouvoir voir les listes d'artistes et acceder à leur description
  • Peut commenter,noter,faire des recherches et se balader librement dans le site
  • La notation et les commentaires se trouvent dans la description d'un album

Un administrateur peut que s'il est connecté

  • Compte admin : admin mdp: admin
  • Dans l'index : A acces a une liste de demande d'inscription et a la liste de tous les commentaires
  • Il peut supprimer les commentaires depuis la vue de l'album ou dans sa liste personnelle de tous les commentaires
  • Dans la liste des demandes d'inscription il peut choisir d'ajouter ou supprimer les demandeurs
